06. 조명 기초

다음 →

왜 조명이 필요한가?

Lesson 01~05에서는 MeshNormalMaterial을 주로 사용했습니다. 이 재질은 조명 없이도 색이 보이지만, 현실감이 없습니다.

실제 세계에서 물체가 보이는 이유는 빛이 반사되기 때문입니다. Three.js에서도 조명을 추가해야 입체감과 현실감이 생깁니다.

Material과 조명의 관계

캔버스 왼쪽 열에서 5가지 Material을 비교할 수 있습니다. 같은 조명 아래에서 Material에 따라 외관이 완전히 달라집니다:

  Material            조명 반응     특징
  ──────────────────────────────────────────
  MeshBasicMaterial   ✗ 무시      항상 동일한 색. 디버그/와이어프레임
  MeshLambertMaterial ○ 확산만    부드러운 음영. 가볍고 빠름
  MeshPhongMaterial   ○ 확산+반짝  하이라이트(shininess). 플라스틱 느낌
  MeshStandardMaterial ◎ PBR     roughness/metalness. 가장 현실적
  MeshToonMaterial    ○ 계단식    셀 셰이딩. 만화/게임 스타일
실무에서는 MeshStandardMaterial이 기본 선택입니다. 물리 기반 렌더링(PBR)으로 가장 현실적인 결과를 제공합니다. MeshBasicMaterial은 조명이 필요 없는 UI, 와이어프레임, 디버그에 사용합니다.

조명 종류

캔버스 오른쪽 열에서 5가지 조명을 비교할 수 있습니다:

AmbientLight — 환경광

모든 면을 균일하게 밝힙니다. 방향이 없으므로 음영이 없습니다.

new THREE.AmbientLight(color, intensity)
// color: 0xffffff (흰색)
// intensity: 0.2 (약하게 — 보조용)

단독 사용하면 밋밋합니다. 다른 조명과 함께 그림자가 너무 어두운 부분을 밝히는 보조 역할로 사용합니다.

DirectionalLight — 방향광

태양광처럼 모든 광선이 평행합니다. 위치보다 방향이 중요합니다.

const light = new THREE.DirectionalLight(0xffffff, 1.5)
light.position.set(1, 3, 1)  // 방향을 결정하는 위치
// 기본 target은 (0,0,0)

PointLight — 점광원

전구처럼 한 점에서 사방으로 빛을 발산합니다. 거리에 따라 약해집니다.

new THREE.PointLight(color, intensity, distance)
// distance: 0이면 무한, 양수면 그 거리까지만 영향

캔버스에서 PointLight가 원형으로 움직이면서 구체를 비추는 것을 확인하세요.

SpotLight — 스포트라이트

손전등처럼 원뿔 형태의 빛을 비춥니다.

new THREE.SpotLight(
  color, intensity, distance,
  angle,     // 원뿔 반각 (라디안)
  penumbra   // 경계 부드러움 (0~1)
)

HemisphereLight — 반구광

하늘색(위) + 땅색(아래)로 자연 환경을 시뮬레이션합니다.

new THREE.HemisphereLight(
  skyColor,    // 위에서 오는 빛 (하늘색)
  groundColor, // 아래에서 오는 빛 (땅색)
  intensity
)

조명 조합의 원칙

  일반적인 조명 조합:

  AmbientLight (약하게)     ← 그림자 영역이 너무 어두운 것 방지
  + DirectionalLight (메인) ← 주요 음영, 그림자 생성
  + PointLight (선택)       ← 포인트 강조, 실내 조명

  또는:

  HemisphereLight          ← 자연스러운 환경광 기본
  + DirectionalLight (메인) ← 태양 역할
조명은 성능 비용이 있습니다. 조명이 많을수록 fragment shader 계산이 늘어나므로, 필요한 만큼만 사용하는 것이 좋습니다. 일반적으로 2~4개의 조명이면 충분합니다.

PBR: roughness와 metalness

MeshStandardMaterial의 핵심 파라미터입니다. 캔버스에서 스크롤하여 5×5 구체 그리드를 확인하세요:

  • roughness (0~1) — 0=거울처럼 매끄러움, 1=완전히 거침
  • metalness (0~1) — 0=비금속(플라스틱, 나무), 1=금속(철, 금)
           metalness →
           0    0.25   0.5   0.75   1
  r  0   [ 플    라    스    틱   금속 ]  매끄러움
  o  .25 [ .     .     .     .     .  ]
  u  .5  [ .     .     .     .     .  ]
  g  .75 [ .     .     .     .     .  ]
  h  1   [ 거    친    돌    녹    슨  ]  거침

roughness=0, metalness=1이면 거울 같은 금속, roughness=1, metalness=0이면 거친 돌/나무 느낌입니다.

LightHelper

각 조명의 위치/방향/범위를 시각화하는 헬퍼입니다:

new THREE.DirectionalLightHelper(light, size)
new THREE.PointLightHelper(light, size)
new THREE.SpotLightHelper(light)
new THREE.HemisphereLightHelper(light, size)

디버깅 시 조명이 어디를 비추는지 확인하는 데 필수적입니다. 프로덕션에서는 제거합니다.

전체 코드

1단계: Material 비교

// 조명 무시 — 항상 같은 색
new THREE.MeshBasicMaterial({ color: 0x4488ff })

// 확산 반사만 — 가볍고 빠름
new THREE.MeshLambertMaterial({ color: 0x4488ff })

// 확산 + 반짝임 — shininess로 하이라이트 크기 조절
new THREE.MeshPhongMaterial({
  color: 0x4488ff,
  shininess: 100,  // 높을수록 작고 날카로운 반짝임
})

// PBR — 가장 현실적
new THREE.MeshStandardMaterial({
  color: 0x4488ff,
  roughness: 0.4,   // 0=매끄러움, 1=거침
  metalness: 0.3,   // 0=비금속, 1=금속
})

// 셀 셰이딩 — 만화풍
new THREE.MeshToonMaterial({ color: 0x4488ff })

MeshBasicMaterial만 조명에 반응하지 않습니다. 나머지는 모두 씬에 조명이 있어야 보입니다.

2단계: AmbientLight

// 모든 면을 균일하게 밝힘 — 방향 없음
const ambient = new THREE.AmbientLight(
  0xffffff,  // 색상
  0.2        // 강도 (보조용이므로 약하게)
)
scene.add(ambient)

3단계: DirectionalLight + Helper

const dirLight = new THREE.DirectionalLight(0xffffff, 1.5)
dirLight.position.set(1, 3, 1)
// position은 "빛이 어디에서 오는지"를 정의
// 실제 광선은 position → target 방향 (기본 target: 원점)
scene.add(dirLight)

// 위치와 방향을 시각화
const helper = new THREE.DirectionalLightHelper(dirLight, 0.5)
scene.add(helper)

DirectionalLight의 position은 "광원의 위치"가 아니라 "빛이 오는 방향"을 결정합니다. 무한히 먼 곳에서 평행하게 오는 빛이므로, position을 (1,3,1)에서 (100,300,100)으로 바꿔도 결과는 동일합니다.

4단계: PointLight (움직이는)

const pointLight = new THREE.PointLight(
  0xff8844,  // 따뜻한 주황색
  3,         // 강도
  8          // 최대 거리 (0=무한)
)
pointLight.position.set(0, 2.5, 0)
scene.add(pointLight)

// animate 내부: 원형으로 이동
pointLight.position.x = Math.sin(t * 1.5) * 1.2
pointLight.position.z = Math.cos(t * 1.5) * 1.2

PointLight를 움직이면 실시간으로 음영이 변합니다. 실내 조명, 촛불, 마법 효과 등에 사용합니다.

5단계: SpotLight

const spotLight = new THREE.SpotLight(
  0x88ff44,       // 색상
  5,              // 강도
  8,              // 최대 거리
  Math.PI / 6,    // 원뿔 반각 (30°)
  0.3             // penumbra (경계 부드러움 0~1)
)
spotLight.position.set(0, 3, 0)
spotLight.target.position.set(0, 0, 0)  // 비추는 방향
scene.add(spotLight)
scene.add(spotLight.target)  // target도 씬에 추가해야 함!

penumbra가 0이면 경계가 날카롭고, 1에 가까울수록 부드럽게 페이드아웃됩니다.

6단계: HemisphereLight

const hemiLight = new THREE.HemisphereLight(
  0x87ceeb,   // 하늘색 (위에서)
  0x362907,   // 갈색 (아래에서 — 땅 반사)
  1.5         // 강도
)
scene.add(hemiLight)

야외 씬에서 AmbientLight 대신 사용하면 더 자연스럽습니다. 위쪽 면은 하늘색, 아래쪽 면은 땅색으로 은은하게 물듭니다.

7단계: roughness / metalness 그리드

for (let r = 0; r < 5; r++) {
  for (let m = 0; m < 5; m++) {
    const mat = new THREE.MeshStandardMaterial({
      color: 0xcccccc,
      roughness: r / 4,    // 0, 0.25, 0.5, 0.75, 1
      metalness: m / 4,    // 0, 0.25, 0.5, 0.75, 1
    })
    const sphere = new THREE.Mesh(
      new THREE.SphereGeometry(0.35, 32, 16), mat
    )
    sphere.position.set(x + m * 1.2, 0.4, z + r * 1.2)
    scene.add(sphere)
  }
}

25개의 구체가 roughness(행)와 metalness(열)의 모든 조합을 보여줍니다. 좌상단은 매끈한 플라스틱, 우하단은 거친 녹슨 금속입니다.
