바람 레이어 시스템

나무의 각 부분이 서로 다른 주파수와 진폭으로 흔들립니다:

  레이어 구조:

  Layer 0: 줄기 흔들림 (trunk sway)
  ├─ 가장 느림, 가장 큰 진폭
  ├─ 전체 나무가 한 방향으로 기울어짐
  └─ 높이² 에 비례 (뿌리=0, 꼭대기=최대)

  Layer 1: 가지 흔들림 (branch sway)
  ├─ 중간 속도, 중간 진폭
  ├─ 각 가지가 개별적으로 흔들림
  └─ 가지 길이 위치에 비례

  Layer 2: 잎 떨림 (leaf flutter)
  ├─ 가장 빠름, 가장 작은 진폭
  ├─ 다중 주파수 (4Hz + 6.7Hz)
  └─ 개별 잎마다 위상 오프셋

바람 파라미터

  • 풍향: 바람이 부는 방향 (라디안)
  • 풍속: 전체 바람 세기
  • 줄기 흔들림: trunk sway 강도
  • 가지 흔들림: branch sway 강도
  • 잎 떨림: leaf flutter 강도
  • 공간 위상: 위치에 따른 바람 위상 차이

구현 방식

vertex shader에서 각 레이어를 합산:

  최종 위치 = 원래 위치
    + trunk_sway(worldPos, time)
    + branch_sway(worldPos, time)
    + leaf_flutter(worldPos, time)

  각 레이어:
    phase = dot(worldPos.xz, windDir) * spatialScale
          + time * speed
    displacement = sin(phase) * strength

vertex color alpha 인코딩

  alpha 0.00~0.25: trunk wind (줄기)
  alpha 0.25~0.50: branch wind (가지)
  alpha 0.50~1.00: leaf flutter (잎)

이 페이지에서는 vertex color 없이 geometry depth로 레이어를 구분합니다.

바람 vertex shader

줄기 흔들림

float trunkPhase = spatialPhase + uTime * speed;
float trunkSway = sin(trunkPhase);
float trunkFactor = heightT * heightT;
transformed.x += windDir.x * trunkSway * strength * trunkFactor;

가지 흔들림

float branchPhase = spatialPhase + uTime * speed * 1.3;
float branchSway = sin(branchPhase);
transformed.x += windDir.x * branchSway * strength * 0.03;

잎 떨림

float leafPhase = hash(worldPos.xz * 20.0) * TAU;
float flutter = sin(uTime * 4.0 + leafPhase)
              + sin(uTime * 6.7 + leafPhase * 1.3) * 0.5;
transformed += flutter * strength * 0.01;
