1단계: 잎 배치 곡선

배치 곡선 설계

가지 위에 잎이 분포하는 패턴을 3개 파라미터로 제어합니다:

  가지 시작                    가지 끝
  │                              │
  │  startT    peakT             │
  │    │         │               │
  │    ╱─────╲   │    ╱──╲      │
  │  ╱         ╲ │  ╱      ╲    │
  │╱             ╲╱          ╲──│
  ├──────────────────────────────┤
  0           t →                1

  startT: 잎이 시작되는 위치
  peakT:  가장 풍성한 위치
  끝:     converge만큼 좁아지며 모임

밀도 곡선 (density)

startT → peakT: smoothstep으로 점점 증가

peakT → 1: converge로 점점 감소 (모이는 느낌)

반경 곡선 (radius)

밀도와 유사하지만 공간적 퍼짐을 제어:

  반경:
  0 ──╱──peak──╲── 0 (수렴)
     ↑          ↑
  startT      converge로 좁아짐

시각화 요소

  • 색상 점: 배치 위치 (빨=시작, 녹=피크, 파=끝)
  • 파란 곡선: 밀도 + 반경 윤곽
  • 초록 곡선: 반대쪽 반경
  • 잎 메시: 실제 잎 배치

시각화 토글로 점/가지/잎/곡선을 켜고 끌 수 있습니다.

조절 가이드

  • 시작 위치: 높이면 가지 끝쪽에만 잎이 생김
  • 피크 위치: 가장 풍성한 지점
  • 끝 수렴도: 높이면 가지 끝이 뾰족하게 모임
  • 퍼짐 반경: 가지에서 얼마나 멀리 퍼지는지
  • 피크 반경 배율: 피크 지점의 볼륨감

배치 곡선 코드

밀도 함수

function getDensity(t) {
  if (t < startT) return 0
  if (t < peakT) {
    // smoothstep 증가
    const s = (t - startT) / (peakT - startT)
    return s * s * (3 - 2 * s)
  } else {
    // converge로 감소
    const s = (t - peakT) / (1 - peakT)
    return 1 - s * s * converge
  }
}

반경 함수

function getRadius(t) {
  if (t < startT) return 0
  if (t < peakT) {
    const s = (t - startT) / (peakT - startT)
    return spreadRadius * s * peakRadiusScale
  } else {
    const s = (t - peakT) / (1 - peakT)
    const taper = 1 - s * converge
    return spreadRadius * peakRadiusScale * taper
  }
}
