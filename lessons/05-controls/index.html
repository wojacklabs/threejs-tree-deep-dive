05. OrbitControls

다음 →

OrbitControls란?

Three.js 코어에는 포함되지 않지만, 공식 addon으로 제공되는 카메라 컨트롤러입니다. 마우스/터치로 카메라를 직관적으로 조작할 수 있게 해줍니다.

import { OrbitControls } from
  'three/examples/jsm/controls/OrbitControls.js'

const controls = new OrbitControls(camera, canvas)
// camera: 조작할 카메라
// canvas: 이벤트를 받을 DOM 요소

기본 조작

지금 바로 캔버스에서 시도해보세요:

  • 좌클릭 드래그 — 회전 (orbit)
  • 우클릭 드래그 — 이동 (pan)
  • 스크롤 — 줌 (dolly)
  • 가운데 클릭 드래그 — 이동 (pan)

캔버스 왼쪽 상단에 현재 distance(거리), polar(수직각), azimuth(수평각)가 표시됩니다. 조작하면서 값이 변하는 것을 확인해보세요.

controls.target

OrbitControls의 핵심 개념입니다. 카메라가 어디를 중심으로 회전하는지를 결정합니다:

           카메라
            ╲│╱
             ●  ← camera.position
             │
             │  distance (거리)
             │
             ○  ← controls.target (회전 중심)

기본값은 (0, 0, 0)이며, 캔버스의 노란 구체가 target 위치입니다.

Random Target 버튼을 눌러 target을 다른 오브젝트로 바꿔보세요. 회전 중심이 바뀌는 것을 체감할 수 있습니다.

controls.target은 camera.lookAt()을 대체합니다. OrbitControls 사용 시 lookAt을 직접 호출하면 충돌합니다. target을 바꾸면 자동으로 lookAt이 갱신됩니다.

Damping (관성)

드래그를 놓았을 때 부드럽게 멈추는 효과입니다:

controls.enableDamping = true
controls.dampingFactor = 0.08  // 0~1, 낮을수록 부드러움

이 효과를 적용하려면 매 프레임 controls.update()를 호출해야 합니다. 호출하지 않으면 damping이 동작하지 않습니다.

제한 설정

사용자가 카메라를 너무 가까이/멀리, 또는 바닥 아래로 보내는 것을 방지할 수 있습니다:

줌 제한:

controls.minDistance = 2    // 최소 거리
controls.maxDistance = 20   // 최대 거리

수직 각도 제한:

// polar angle: 0=위에서 내려다봄, π=아래에서 올려다봄
controls.minPolarAngle = 0
controls.maxPolarAngle = Math.PI * 0.85  // 바닥 아래 방지
  polar = 0       polar = π/2      polar = π
  (위에서)         (수평)            (아래에서)

    👁               👁
    │             ───●───           │
    ▼                               ▲
    ●                               ●
                                    👁

수평 각도 제한:

// 특정 방향으로만 회전 허용 (건축 시각화 등)
controls.minAzimuthAngle = -Math.PI / 4  // -45°
controls.maxAzimuthAngle = Math.PI / 4   // +45°

자동 회전

사용자 조작이 없을 때 카메라가 자동으로 회전합니다:

controls.autoRotate = true
controls.autoRotateSpeed = 2.0  // 기본값 2.0

캔버스의 Auto Rotate 버튼으로 켜고 끌 수 있습니다. 제품 전시, 쇼케이스 등에서 많이 사용됩니다.

기능 비활성화

특정 조작을 막을 수 있습니다:

controls.enableRotate = false  // 회전 비활성화
controls.enableZoom = false    // 줌 비활성화
controls.enablePan = false     // 이동 비활성화
controls.enabled = false       // 전체 비활성화

예: 제품 뷰어에서 pan을 막아 오브젝트가 화면 밖으로 이동하지 않게.

reset()

controls.reset()은 카메라를 초기 위치/방향으로 되돌립니다. OrbitControls 생성 시점의 상태가 기록됩니다.

캔버스의 Reset 버튼으로 확인해보세요.

다른 컨트롤러들

OrbitControls 외에도 목적에 따라 다른 컨트롤러를 사용합니다:

  • FlyControls — 비행 시뮬레이터 방식 (6자유도)
  • PointerLockControls — FPS 게임 방식 (마우스 잠금)
  • TrackballControls — OrbitControls와 유사하나 수직 제한 없음
  • MapControls — 지도 앱 방식 (좌클릭=pan, 우클릭=회전)
  • DragControls — 오브젝트를 드래그로 이동

전체 코드

1단계: import와 생성

import { OrbitControls } from
  'three/examples/jsm/controls/OrbitControls.js'

const controls = new OrbitControls(camera, canvas)
// camera: PerspectiveCamera 인스턴스
// canvas: <canvas> DOM 요소 (이벤트 리스너 대상)

두 번째 인자가 renderer.domElement가 아니라 직접 canvas를 넘기는 점에 주의하세요. Vite에서는 canvas를 직접 HTML에 만들어두므로 동일합니다.

2단계: 옵션 설정

// 관성 효과
controls.enableDamping = true
controls.dampingFactor = 0.08

// 줌 범위 제한
controls.minDistance = 2
controls.maxDistance = 20

// 바닥 아래로 못 가게
controls.maxPolarAngle = Math.PI * 0.85

// 자동 회전 (기본 off)
controls.autoRotate = false
controls.autoRotateSpeed = 2.0

생성 직후에 옵션을 설정합니다. dampingFactor가 낮을수록 더 부드럽게(느리게) 멈춥니다.

3단계: 렌더 루프에서 update()

function animate() {
  requestAnimationFrame(animate)

  controls.update()  // ← 필수! damping/autoRotate에 필요
  renderer.render(scene, camera)
}
animate()

controls.update()를 빠뜨리면:

  • damping이 동작하지 않음 (즉시 멈춤)
  • autoRotate가 동작하지 않음

damping이나 autoRotate를 사용하지 않으면 update()를 생략해도 되지만, 항상 호출하는 습관을 들이는 것이 좋습니다.

4단계: target 변경

// target을 특정 오브젝트 위치로 변경
controls.target.copy(someObject.position)
// 이후 자동으로 카메라가 해당 위치를 중심으로 회전

// 또는 직접 좌표 지정
controls.target.set(2, 0, -1)

target을 바꾸면 카메라의 lookAt 방향이 자동으로 갱신됩니다.

5단계: 상태 정보 읽기

// 카메라와 target 사이의 거리
const dist = camera.position.distanceTo(controls.target)

// 수직 각도 (0=위에서, π/2=수평, π=아래에서)
const polar = controls.getPolarAngle()

// 수평 각도 (-π ~ π)
const azimuth = controls.getAzimuthalAngle()

캔버스 왼쪽 상단 info bar에 이 값들을 실시간 표시하고 있습니다. 조작하면서 값의 변화를 확인해보세요.

6단계: 버튼 이벤트

// 초기 상태로 복원
document.getElementById('btn-reset')
  ?.addEventListener('click', () => {
    controls.reset()
  })

// 자동 회전 토글
document.getElementById('btn-auto')
  ?.addEventListener('click', () => {
    controls.autoRotate = !controls.autoRotate
  })

// target을 랜덤 오브젝트로 변경
document.getElementById('btn-target')
  ?.addEventListener('click', () => {
    const obj = objects[Math.floor(Math.random() * objects.length)]
    controls.target.copy(obj.position)
  })

리사이즈 패턴 (반복)

function resize() {
  const parent = canvas.parentElement!
  const w = parent.clientWidth
  const h = parent.clientHeight
  renderer.setSize(w, h)
  renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2))
  camera.aspect = w / h
  camera.updateProjectionMatrix()
}
window.addEventListener('resize', resize)
resize()
