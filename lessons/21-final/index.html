Scroll 애니메이션

스크롤 위치에 따라 카메라 Y를 이동하여 각 섹션의 3D 오브젝트가 순서대로 보이게 합니다:

scrollContainer.addEventListener('scroll', () => {
  scrollY = scrollContainer.scrollTop
})

// animate 내부:
const targetY = -(scrollY / window.innerHeight) * 8
camera.position.y = THREE.MathUtils.lerp(
  camera.position.y,
  targetY + 2,
  0.05  // 부드러운 보간
)

MathUtils.lerp로 보간하면 스크롤이 부드럽게 따라갑니다. 0.05는 보간 속도 — 낮을수록 더 부드럽고 느림.

HTML + WebGL 혼합

캔버스를 투명 배경으로 설정하고, HTML 요소를 위에 겹쳐 놓는 방식입니다:

// 투명 캔버스
const renderer = new THREE.WebGLRenderer({
  canvas,
  antialias: true,
  alpha: true,        // ← 핵심!
})
renderer.setClearColor(0x000000, 0)
// 두 번째 인자 0 = 완전 투명
  레이어 구조:

  ┌──────────────────────┐
  │  HTML 카드 (z-index 높음)│  pointer-events: auto
  ├──────────────────────┤
  │  3D Canvas (중간)     │  alpha: true
  ├──────────────────────┤
  │  배경 CSS (낮음)      │  background gradient
  └──────────────────────┘

pointer-events를 적절히 설정하여 HTML 카드는 클릭 가능하고, 빈 영역은 3D 조작이 가능하게 합니다.

Mouse Parallax

마우스 위치에 따라 카메라가 미세하게 움직여 깊이감을 줍니다:

window.addEventListener('mousemove', (e) => {
  mouseX = (e.clientX / window.innerWidth - 0.5) * 2
  mouseY = (e.clientY / window.innerHeight - 0.5) * 2
})

// animate:
camera.position.x = THREE.MathUtils.lerp(
  camera.position.x,
  mouseX * 0.5,
  0.05
)

스크롤 → 카메라 이동

const scrollContainer = document.getElementById('scroll')
let scrollY = 0

scrollContainer.addEventListener('scroll', () => {
  scrollY = scrollContainer.scrollTop
})

function animate() {
  // 스크롤 비율 → 카메라 Y
  const targetY = -(scrollY / innerHeight) * 8
  camera.position.y = THREE.MathUtils.lerp(
    camera.position.y, targetY + 2, 0.05
  )
}

각 섹션의 3D 오브젝트는 Y축으로 8유닛 간격으로 배치. 스크롤 1 섹션 = 카메라 Y -8 이동.

마우스 패럴랙스

let mouseX = 0, mouseY = 0
window.addEventListener('mousemove', (e) => {
  mouseX = (e.clientX / innerWidth - 0.5) * 2
  mouseY = (e.clientY / innerHeight - 0.5) * 2
})

// animate:
camera.position.x = lerp(cam.x, mouseX * 0.5, 0.05)
camera.rotation.x = lerp(cam.rx, -mouseY * 0.05, 0.05)
